Getidmis, the INN IDMIS Downloader

The IDMIS Downloader (getidmis) is available for Linux, Windows and Mac.

Since most people aren't comfortable with the command line, there are simple options to make it easy for you to install and use. However, since getidmis is essentially a command line program, if you are happy using the command line, you can just download the software and run it.

For Windows and Mac users, I have made a simple installer and a clickable way of running the program.

Installation and Usage Options

The following options are available for installation and usage:

  • Windows (If you don't know how to use the command line)
  • Mac (If you don't know how to use the command line)
  • Command line (Linux, Windows, Mac)

The code

If you are a programmer and wish to check how thr program works or you want to modify it and compile it yourself, the program is written in C and the code can be obtained from  GitHub .

The code depends in our BiopLib, our C library of useful functions which you will need to build and install first. This can also be obtained from  GitHub .
