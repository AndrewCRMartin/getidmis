Getidmis, the INN IDMIS Downloader

Mac Installation

Before you start...

getidmis makes use of a program called cURL to grab files from the internet. It requires at least version 8 of this software to handle the password protected certificate files. To see what version you have, open a command prompt and type curl --version If you can upgrade to the latest version before installing getidmis, things will definitely go smoothly! If you can't, then the installer will try to work out your hardware architecture and install a local copy of the correct version. Since I don't have a recent Mac, I cannot test this!

Obtaining the getidmis installer...

  1. Click on this link:  GetIDMIS Installer  to download it - I am assuming you will download it to your Downloads folder.
  2. Depending on your browser, you may obtain an alert saying something like 'Insecure download blocked'. Click the keep option.
  3. With Chrome or Edge browsers, the download may get 'stuck' as a Unconfirmed xxxxx.crdownload file. If this is the case, click the download progress button and click the Keep button. Alternatively, go to the Downloads folder and rename the file to install.sh

Note: The dire warnings about 'insecure downloads' really can be safely ignored! They are not suggesting any security issues with the file itself, they are simply saying that it is downloaded over HTTP rather than HTTPS (which encrypts the file as it is being downloaded).

Running the getidmis installer...

  1. Double-click on install.sh in your Downloads folder.
  2. The Mac may respond with an alert box warning about running untrusted software. Click whatever you need to, to run it anyway!
  3. The installer will prompt you for your certificate password (as supplied by Antonio).
  4. It will create a getidmis folder in your home folder (below I refer to that new folder as Home...getidmis) and save your certificate password there. It will then download the actual getidmis software and the newer versions of the curl program. As above, if you can make sure that curl V8.0 (or higher) is installed before you start, then getidmis will use that.
  5. The installer will also prompt you to install the certificate file (as supplied by Antonio) in the getidmis folder. Copy this file (typically called surname-cert.p12, from wherever you downloaded it, into the Home...getidmis folder and rename it to cert.p12.
  6. Finally, the installer will tell you to press Enter to indicate you have finished.

Running getidmis on the Mac

  1. In the Home...getidmis folder, there is a file called getidmis.sh. Copy this file to wherever you want to download the files from IDMIS.
    For example, if you are downloading the files for PL132, you might have created a folder called Documents/PL132, so copy getidmis.sh to that folder.
  2. Double click on the copy of getidmis.sh that you just made. Files will be downloaded from IDMIS to that folder.
  3. You then enter the request numbers for the document you wish to download. You can enter one number per line, or multiple numbers on a single line separated by spaces - do not use any other characters such as commas or semi-colons!
  4. When you are done entering numbers, simply press Enter another time to enter a blank line.
  5. All the files associated with each request number will then be downloaded!
